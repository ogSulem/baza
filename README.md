# Telegram бот: база поставщиков и заказчиков

## Запуск

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Создайте `.env` по примеру `.env.example` и заполните:

- `BOT_TOKEN`
- `ADMIN_IDS` (через запятую)
- `DB_PATH` (по желанию)

3. Запустите бота:

```bash
python main.py
```

## Админка

- `/admin` — меню админа
  - выгрузка поставщиков/заказчиков в CSV
  - управление категориями (добавить/переименовать/вкл-выкл/удалить)
